 What are the different types of css?
 -----------------------------------------
 There are 3 types of CSS presents which mentioned below.
 1. inline css :
 ------------------------------
 Inline css written inside the tag.
 Example;
 ---------------
 <h1 style="color:orange">ByteCode</h1>
----------------------------------------------------

 2. internal css:
 --------------------------
 This type of css can written inside the head section of single page.
 by using tag called as <style></style>
----------------------------------------------------
3. external css (famous):
------------------------------
This types of css can be written inside external file which stored by using the extension called as .css 
When we create this file, we need to link this file with our html page, by using <link> tag.
<link> tag present inside <head></head>.

How we can link one css file to another css file?
---------------------------------------------------------
By using @import keyword 

Example:
----------------------------
I have two css files 
style.css
style1.css

I want to add style1.css code inside style.css file
@import "style1.css"

What is mean by selectors and what are the different types of selectors?
--------------------------------------------------------------------------
Def : selectors are the targetting elements.

What are the different types of selectors?
----------------------------------------------------
1. id selectors:
-----------------------
Id selectors are unique. Id can represented by using # symbol.

The combination of padding and Margin is called css box model
Padding: clockwise direction --> top, right, bottom, left
Margin: clockwise direction --> top, right, bottom, left

Padding : Inside the space between border.
padding-right, padding-left, padding-top, padding-bottom
padding: 10px 20px  --> 10px --> top bottom and 20px --> left right
padding:10px --> padding set in all direction.
Margin : Outside space of border.
-------------------------------------------------------------------------

2. class selectors:
-----------------------------
Class selectors can be denoted by dot(.). We can use multiple classes for single element.

3. tag selectors:
-----------------------------
We can use html element directly for targetting. 
Example:
----------------
h1, img, div, p, table, tr, td

4. class with tag selectors:
---------------------------------
